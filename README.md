# Psychic Source — The Celestial Atelier

A luxury reimagining of psychicsource.com as a heritage house where **everything emerges from a single point of source-light.** Deep violet-indigo cosmos, champagne gold, Cormorant Garamond + Jost.

This package is the complete build: a static, self-contained website, 150+ bespoke generated images, the full design system, and a reusable brand skill so you can keep building on-brand in Claude Code.

## Quick start
```
python3 -m http.server 8000
# open http://localhost:8000/index.html
```
No build step, no dependencies.

## Working with it in Claude
Open this folder in Claude Code. The `celestial-atelier` skill loads automatically and teaches Claude the whole system, so new pages and components come out on-brand. Start at [`CLAUDE.md`](CLAUDE.md).
